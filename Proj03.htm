PROGRAM SPECIFICATIONS

Write a program named Proj03 (that uses the driver class definition provided in the file named Proj03.java) that can be used to connect to, download, render, and navigate simple web pages as shown in Image01, Image02, and Image03. (By simple, I mean web pages that do not contain frames, automatic forwarding, and other fancy HTML features.)

Your program must provide a Back button and a Forward button as shown in Image01. You can navigate through simple web pages by clicking hyperlinks with the mouse. You can backtrack your navigation path using the Back button and then move forward through your navigation path using the Forward button.

You can navigate to a particular web page by entering the URL in the text field shown at the top of Image01. (Note that if the URL is too long to fit in the text field, part of the URL won't be visible.)

To make it easier for me to to test and grade your assignment, cause your program to default to the following web page at startup by automatically entering the following URL in the text field and by automatically opening the corresponding web page in your output:

http://www.dickbaldwin.com/tocdsp.htm

Once there, you should be able to navigate to any of the following simple web pages:

100 Periodic Motion and Sinusoids
104 Sampled Time Series
108 Averaging Time Series

(You may be able to navigate to other web pages as well, but many of the links on that page point to pages that contain HTML features that your program is not required to support.)

Your program output must provide scroll bars when needed as shown in Image01.

You must be able to use hyperlinks, the mouse, the mouse wheel, the scroll bars, the Back button, and the Forward button to expose the material shown in Image01, Image02, and Image03.

You must also display the following text on the command-line screen.

I certify that this program is my own work
and is not the work of others. I agree not
to share my solution with others.
Print your name here.

Submit only the source code file named Proj03Runner.java.
